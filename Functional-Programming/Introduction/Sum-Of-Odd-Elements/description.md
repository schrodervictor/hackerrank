## Constraints

The list will have 1-100 elements. Each element will be an integer X such that -100&lt;=X&lt;=100.

2
4
6
5
7
8

1
```

## Sample Output

```

## Explanation

Sum of odd elements = 3 + 5 + 7 + 1 = 16
